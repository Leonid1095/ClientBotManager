# Telegram-бот для приёма заказов на разработку

## Возможности бота

### Главное меню:
- **Заказать бота** — многошаговая форма сбора информации о заказе
- **Портфолио** — примеры работ с кнопкой "Посмотреть кейс"
- **FAQ** — часто задаваемые вопросы
- **Чат поддержки** — прямое общение с разработчиком
- **Калькулятор стоимости** — расчёт стоимости по параметрам
- **Статус заказа** — проверка статуса тикета по номеру
- **О компании** — информация о разработчике
- **Связаться с разработчиком** — контакты
- **Отзывы** — просмотр и добавление отзывов
- **Бонусы и рефералы** — реферальная программа с бонусами

### Форма заказа:
1. ФИО клиента
2. Контактные данные
3. Идея для бота
4. Тип бота (чат-бот, магазин, интеграция и т.д.)
5. Бюджет
6. Желаемые сроки
7. Тариф/опции
8. Особые настройки
9. Прикрепление файлов
10. Выбор хостинга (ваш/мой сервер)
11. Использование бонусов (если есть)
12. Подтверждение заявки

### Автоматизация:
- **Тикет-система:** каждая заявка формируется как тикет с уникальным ID
- **Статусы:** новый, в работе, ожидает ответа, завершено (настраивается в Google Sheets)
- **Хранение:** интеграция с Google Sheets для тикетов
- **Уведомления:** админ получает заявки в личные сообщения

### Бонусы и рефералы:
- Реферальная ссылка для каждого пользователя
- Бонус 100 руб. за каждого приглашённого
- Автоматическое применение бонусов при заказе
- Списание бонусов после подтверждения заказа
- Калькулятор стоимости учитывает бонусы

### Верификация:
- Идентификация по Telegram ID (уникальный идентификатор)
- Сохранение user_id для каждого тикета

## Настройка и запуск

### 1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

### 2. Настройка Google Sheets (опционально):
- Создайте Google Sheet с названием `BotOrders`
- Получите credentials.json и сохраните как `google-credentials.json`
- Настройте доступ к Google Sheets API

### 3. Конфигурация:
Откройте `config.py` и заполните:
```python
TELEGRAM_TOKEN = "ваш_токен_бота"
ADMIN_USER_ID = 123456789  # ваш user_id
```

### 4. Запуск:
```bash
python bot.py
```
